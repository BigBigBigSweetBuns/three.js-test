## three.js 学习

#### 启动

##### 启动指令：

打包： `npm build`

监听打包： `npm run build-watch`

##### 开发启动顺序：

1. 监听打包
2. 使用 `Live Server` 监听打包后的文件夹

##### 环境：

node.js 

webpack 打包

vscode 插件 `Live Server` 作为本地服务器

##### 环境配置：

`npm i` 安装环境

`Live Server` 启动指向打包后 `index.html` 文件
